# Intent Example - Retro Operator Shell

## Direction

- Surface: `apps/web/frontend/app/retro`
- Audience: operator and analyst
- Delivery lane: publish
- Density: compact
- Motion: subtle

## Taste brief

Build a retro-inspired admin and end-user UI for homelab-analytics.

- Mood: terminal-adjacent, warm industrial, analytical rather than playful
- Priorities: dense information display, calm contrast, clear hierarchy, fast keyboard-friendly interactions
- Anti-goals: neon cyberpunk, glassmorphism, oversized marketing-landing styling

## Reference adjectives

- restrained
- trustworthy
- operational
- tactile

## Priority screens

- operating picture overview
- money detail
- utilities detail
- operations detail
- source onboarding and remediation

## Acceptance cues

- readable in daylight without glow effects doing the work
- numerics feel precise and easy to compare
- operator actions are obvious without turning the shell into a marketing page
- alternate renderer feels deliberate, not like a novelty skin
