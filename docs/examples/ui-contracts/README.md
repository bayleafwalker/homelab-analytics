# UI Contract Example Bundle

This directory contains the minimum artifact set for a publish-lane UI direction:

- `intent.md` - human-written taste brief
- `baseline.tokens.json` - agent-generated, human-approved style baseline
- `ui-contract.yaml` - primitive, scenario, and validation contract

Use these files as skeletons when a new web shell, alternate renderer, or stable UI theme needs a reviewable contract before broad implementation.
